# Network Troubleshooting Lab

## Objective
Troubleshoot common network connectivity issues in Windows.

## Tasks Performed
- Checked IP configuration using ipconfig
- Tested connectivity using ping
- Flushed DNS cache
- Restarted network adapter
- Verified internet access

## Tools Used
- Command Prompt
- ipconfig
- ping
- Windows Network Settings

## Outcome
Successfully identified and resolved network connectivity issues.
